Pitch Class Router

MIDI bridge from Scale Navigator to pitch-class-aware plugins.

Converts scale and chord data into MIDI Control Change messages, automatically synchronizing pitch-class-aware audio plugins with Scale Navigator. Eliminates the tedious workflow of manually updating 12 pitch-class toggles across multiple plugins during performance and composition.

2024 Software
JUCE / C++ / WebMIDI

Two Implementations:

Native JUCE plugin (VST3, AU, Standalone)
Lightweight web browser version using WebMIDI API

Compatible Plugins:

Fluid Pitch (Pitch Innovations)
Chroma (Xynth Audio)
Scaler EQ (Plugin Boutique)
Temperance Pro (Eventide)
Auto-Tune (Antares)
Any plugin with MIDI-mappable pitch-class parameters

Features:

Configurable MIDI input/output ports and channels
12-button MIDI learn helper for each pitch class
Settings persistence with DAW projects
Universal binary for Apple Silicon and Intel
Real-time sync as scales and chords change
